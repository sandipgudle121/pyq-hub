# PYQHub

PYQHub is a simple and responsive web application that helps students quickly find and download Previous Year Question Papers (PYQs) for their courses. Built using **HTML**, **CSS**, and **JavaScript**, the project focuses on providing a clean, fast, and user-friendly interface without relying on any frontend frameworks.

The website allows students to browse question papers by university, branch, semester, subject, and examination session. A search interface and filtering system make it easy to locate relevant papers, while each paper has a dedicated detail page with metadata, PDF preview, and download options.

## Features

* Clean and responsive user interface
* Browse question papers by university, branch, semester, and subject
* Search papers by subject, course code, university, or year
* Filter and sort available question papers
* Paper detail page with metadata and PDF preview
* Direct PDF download support
* Drag-and-drop PDF upload interface (UI)
* Toast notifications for user actions
* Mobile-friendly design

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## Current Status

This project is currently a frontend prototype. The interface and navigation are complete, while features such as real search, filtering, PDF uploads, dynamic data loading, and backend integration are planned for future development.

## Future Improvements

* Dynamic paper data using JSON or a database
* Fully functional search and filtering
* PDF preview and download tracking
* Backend integration for uploads and storage
* Admin dashboard for managing question papers
* Dark mode
* Bookmark favorite papers
* SEO-friendly URLs

## Goal

The goal of PYQHub is to provide students with a centralized platform where they can easily access previous year question papers for exam preparation, reducing the time spent searching across multiple websites or college groups.
