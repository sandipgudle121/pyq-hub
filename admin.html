Admin Helper
This page is intentionally simple. Put the PDF in the right folder, then add one JSON item to data/papers.json for either BSc or MSc.
⬇ Download JSON ↩ Open site

What to add

TitlePaper name
ProgramBSc, MSc or BTech
CodeSubject code
UniversityYour university
SemesterNumeric semester
YearExam year

Where to place files

PDF folderpdfs/bsc/... pdfs/msc/... pdfs/btech/...
JSON filedata/papers.json
Detail pagedetail.html?id=...
Browse listAuto rendered
DownloadUse real PDF path
AuthNot needed
